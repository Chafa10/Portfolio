Proyectos

Gestion de articulos

Software de Gestion de articulos, en el cual podras hacer altas, bajas, modificacion y consultas de productos. Lenguaje C# winform y SQL.

Calculos estadisticos

Este programa te permite mediante el ingreso de datos agrupados o desagrupados, calcular la frecuencia, frecuencia acumulada, variancia, promedio, moda, mediana y te permite graficarlo en un grafico de barras. Lenguaje C# winform.

Calculo de rafagas de un CPU

Software que calcula las rafagas de una CPU para diferentes algoritmos de planificacion de procesos, que estima cuanto tiempo tomara ejecutar una serie de procesos o tareas. Lenguaje C# winform.

Pagina web Pokedex

Pagina web que simula ser una pokedex con abm de pokemones con sus tipos y evoluciones. Lenguajes C#.NET, ASP.NET, SQL, HTML, CSS Y BOOTSTRAP.

Desarrollo de Apis de un Bazar

Aplicacion web(Back-End) que permite mediante peticiones HTTP acceder a los diferentes endpoint que permitiran hacer un ABML de los diferentes productos del bazar. Hecho en Java con framework spring boot, JPA-Hibernate, MySql y postman.
